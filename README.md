# bundle-webpack4

> Верстка сайта (name website) . Используется [webpack4-BEM](https://github.com/SDim87/bundle-webpack4).

## Установка и запуск

```bash
# установка модулей
npm install

# веб сервер с автоперезагрузкой по адресу localhost:8080
npm run dev

# сборка проекта в директорию dist
npm run build

```
